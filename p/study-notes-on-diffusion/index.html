æ›´å…·ä½“çš„ï¼ŒDDPMså°†å…¶å®šä¹‰ä¸ºï¼ˆå·²çŸ¥çš„ï¼‰å‰å‘åŠ å™ªè¿‡ç¨‹(Forward Process)ï¼Œä»–æ˜¯ä¸€ä¸ªMarkovè¿‡ç¨‹ï¼š

å…¶ä¸­ $\beta_t$ ä¹Ÿæ˜¯å·²çŸ¥çš„ï¼Œé¢„å®šä¹‰å¥½çš„ä¸€ä¸ªæ•°åˆ—ï¼Œæ»¡è¶³ $\beta_1 < \beta_2 < \cdots < \beta_T$ï¼Œå¯ä»¥æ˜¯linear schedule, cosine scheduleç­‰ç­‰…

è¿™ä¸ªç­‰å¼ç›¸æ¯”æ¦‚ç‡åˆ†å¸ƒè€Œè¨€æ›´ç›´è§‚çš„ä½“ç°åŠ å™ªè¿‡ç¨‹ï¼šæ¯æ¬¡åœ¨ä¸Šä¸€æ­¥çš„çŠ¶æ€ $x_{t-1}$ ä¸Šï¼Œä¸æ–°å¼•å…¥çš„ä¸€ä¸ªéšæœºå™ªå£° $\varepsilon_t$ è¿›è¡Œçº¿æ€§ç»„åˆï¼Œä»¥æ­¤å¾—åˆ°ä¸‹ä¸€ä¸ªæ—¶åˆ»çš„çŠ¶æ€ $x_{t}$ã€‚ ç°åœ¨æ¥çœ‹çœ‹è¿™ä¸ªMarkové“¾æœ‰ä»€ä¹ˆå¾ˆå¥½çš„æ€§è´¨ï¼šä»»ä½•æ—¶åˆ»çš„ $x_t$ å¯ä»¥ç”± $x_0$ å’Œ $\beta_t$ è¡¨ç¤ºï¼š

$$ \begin{aligned} x_t &= \sqrt{1-\beta_t} x_{t-1} + \sqrt{\beta_t} \varepsilon_t \\ &= \sqrt{\alpha_t} x_{t-1} + \sqrt{1-\alpha_t} \varepsilon_t \\ &= \sqrt{\alpha_t} (\sqrt{\alpha_{t-1}} x_{t-2} + \sqrt{1-\alpha_{t-1}} \epsilon_{t-1}) + \sqrt{1-\alpha_t} \varepsilon_t \\ &= \sqrt{\alpha_t \alpha_{t-1}} x_{t-2} + \sqrt{\alpha_t(1-\alpha_{t-1}) + 1-\alpha_t} \varepsilon_{t-2} \\ &= \sqrt{\alpha_t \alpha_{t-1}} x_{t-2} + \sqrt{1 - \alpha_t\alpha_{t-1}} \varepsilon_{t-2} \\ &= \cdot\cdot\cdot \\ &= \sqrt{\alpha_t \alpha_{t-1}\cdot\cdot\cdot\alpha_1} x_0 + \sqrt{1 - \alpha_t\alpha_{t-1}\cdot\cdot\cdot\alpha_1} \varepsilon \\ &= \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \varepsilon, \quad \bar{\alpha}_t = \prod^t_{i=1} \alpha_i \end{aligned} $$

è¿™é‡Œæˆ‘ä»¬ç®€è®° $\alpha_t = 1 - \beta_t$ï¼Œå¹¶ä¸”å¯¹éšæœºå™ªå£°çš„åˆå¹¶ä¸­ç”¨åˆ°äº†æ­£æ€åˆ†å¸ƒçš„æ€§è´¨ï¼š$\mathcal{N}(\mu_1, \sigma_1^2 I) + \mathcal{N}(\mu_2, \sigma_2^2 I) = \mathcal{N}(\mu_1+\mu_2, (\sigma_1^2 + \sigma_2^2)I)$ã€‚å¦‚æœå†ç”¨æ¦‚ç‡æ¥è¡¨ç¤º $x_t$ å’Œ $x_0$ ä¹‹é—´çš„å…³ç³» $x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \varepsilon$ï¼Œä¹Ÿå°±æ˜¯ä¸‹é¢è¿™ä¸ªæ¡ä»¶æ¦‚ç‡ï¼š

$$ q(x_t \mid x_0) = \mathcal{N} (x_t; \sqrt{\bar{\alpha}_t} x_0, (1 - \bar{\alpha}_t)I) $$

åå‘è¿‡ç¨‹ Reverse Process

å‰å‘è¿‡ç¨‹çš„æ€§è´¨å°±å…ˆæŒ–åˆ°è¿™é‡Œï¼Œæ¥ä¸‹æ¥æˆ‘ä»¬æ¥è€ƒè™‘åå‘è¿‡ç¨‹(Reverse Process)ã€‚çº¦å®šåå‘è¿‡ç¨‹ä¸­çš„æœªçŸ¥æ¦‚ç‡åˆ†å¸ƒç”¨ $p$ æ¥è¡¨ç¤ºï¼ˆä¸å‰å‘è¿‡ç¨‹å·²çŸ¥çš„åˆ†å¸ƒåŒºåˆ†ï¼‰ï¼Œæˆ‘ä»¬å¸Œæœ›èƒ½å¤Ÿåè½¬ä¸Šè¿°ï¼ˆå‰å‘ï¼‰è¿‡ç¨‹ï¼Œå¦‚æœå¯ä»¥ä» $p(x_{t-1} \mid x_t)$ ä¸­é‡‡æ ·ï¼Œé‚£ä¹ˆæˆ‘ä»¬å°±å¯ä»¥åœ¨ $T$ æ—¶åˆ»ä»éšæœºå™ªå£°ï¼ˆæ ‡å‡†æ­£æ€åˆ†å¸ƒï¼‰ä¸­é‡‡æ ·ä¸€ä¸ªåˆå§‹å€¼ $x_T \sim \mathcal{N}(0, I)$ï¼Œç„¶åä¸€æ­¥æ­¥å»å™ªæ¥è¿˜åŸï¼Œæœ€ç»ˆå¾—åˆ°ä¸€ä¸ªä¸çœŸå®å›¾åƒåˆ†å¸ƒï¼ˆè¿‘ä¼¼ï¼‰ä¸€è‡´çš„ $\hat{x}_0$ã€‚

ä½†æ˜¯æˆ‘ä»¬å¹¶æ²¡æœ‰ $p(x_{t-1} \mid x_t)$ çš„å…·ä½“è¡¨è¾¾å¼ï¼Œæ‰€ä»¥æˆ‘ä»¬è®¾è®¡ç¥ç»ç½‘ç»œ(Neural Network)æ¥å­¦ä¹ è¿™ä¸ªåˆ†å¸ƒã€‚

å®šä¹‰ï¼š$p_{\theta}(x_{t-1} \mid x_t) = \mathcal{N} (x_{t-1}; \mu_{\theta}(x_t, t), \Sigma_{\theta}(x_t, t)I)$ï¼Œä¹Ÿå³å‡å€¼å’Œæ–¹å·®åˆ†åˆ«ä¸º $\mu_{\theta}(x_t, t)$ å’Œ $\Sigma_{\theta}(x_t, t)$ è¿™ä¸¤ä¸ªç¥ç»ç½‘ç»œé¡¹ã€‚

æˆ‘ä»¬å…ˆè€ƒè™‘å¦‚ä½•è®¾è®¡Training Lossï¼šDDPMsé€‰æ‹©çš„æ˜¯ä¼˜åŒ–è´Ÿå¯¹æ•°ä¼¼ç„¶(negative log-likelihood)ï¼Œå…ˆä¸Šç»“è®º:

$$ \begin{aligned} -\log p_\theta(x_0) & \leq\mathbb{E}_{q(x_{1:T} |x_0)}\left[-\log\frac{p_\theta(x_{0:T})}{q(x_{1:T}|x_0)}\right] \\ &= \mathbb{E}_{q(x_{1:T}|x_0)}{\left[\underbrace{D_{\mathrm{KL}}(q(x_{T}|x_{0})\|p(x_{T}))}_{L_{T}}+\sum_{t=2}^{T}\underbrace{D_{\mathrm{KL}}(q(x_{t-1}|x_{t},x_{0})\|p_{\theta}(x_{t-1}|x_{t}))}_{L_{t-1}}+\underbrace{\log p_{\theta}(x_{0}|x_{1})}_{L_{0}}\right]} \\ &\triangleq \mathcal{L} \end{aligned} $$

# === Log likelihood calculation ===
def _vb_terms_bpd(self, denoise_fn, x_start, x_t, t, *, clip_denoised: bool, return_pred_xstart: bool):
	true_mean, _, true_log_variance_clipped = self.q_posterior_mean_variance(x_start=x_start, x_t=x_t, t=t)
	model_mean, _, model_log_variance, pred_xstart = self.p_mean_variance(
		denoise_fn, x=x_t, t=t, clip_denoised=clip_denoised, return_pred_xstart=True)
	kl = normal_kl(true_mean, true_log_variance_clipped, model_mean, model_log_variance)
	kl = nn.meanflat(kl) / np.log(2.)

	decoder_nll = -utils.discretized_gaussian_log_likelihood(
		x_start, means=model_mean, log_scales=0.5 * model_log_variance)
	assert decoder_nll.shape == x_start.shape
	decoder_nll = nn.meanflat(decoder_nll) / np.log(2.)

	# At the first timestep return the decoder NLL, otherwise return KL(q(x_{t-1}|x_t,x_0) || p(x_{t-1}|x_t))
	assert kl.shape == decoder_nll.shape == t.shape == [x_start.shape[0]]
	output = tf.where(tf.equal(t, 0), decoder_nll, kl)
	return (output, pred_xstart) if return_pred_xstart else output
